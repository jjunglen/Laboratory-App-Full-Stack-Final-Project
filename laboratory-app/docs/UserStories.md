# User Stories — The Laboratory

---

## MVP (must have for launch)

1. As a user, I want to create an account so that I can save my sneaker preferences securely.
2. As a user, I want to log in so that I can access my personalized sneaker feed.
3. As a user, I want to set my preferred brands, sizes, and price range so that I only see sneakers that match my taste.
4. As a user, I want to view a feed of sneakers that match my preferences so that I can find my next pair quickly.
5. As a user, I want to click a Buy button on a sneaker so that I can purchase the sneakers.
6. As a user, I want to update my preferences so that my feed stays relevant as my taste changes.
7. As a user, I want to receive an in-app notification when a sneaker matches my preferences so that I never miss a drop.
8. As a user, I want to see a notification bell with an unread count so that I know when something new has dropped.
9. As a user, I want to stay logged in so that I don't have to log in every time I open the app.
10. As a user, I want to log out so that my account stays secure when I'm done.

---

## Core but Slightly Expanded

11. As a user, I want to reset my preferences so that I can start fresh without deleting my account.
12. As a user, I want to filter my sneaker feed by brand so that I can quickly browse a specific brand.
13. As a user, I want to mark notifications as seen so that my unread count stays accurate.

---

## Nice to Haves

14. As a user, I want to view my profile so that I can see my account info and activity at a glance.
15. As a user, I want to see how many matches and alerts I have so that I can track my activity over time.
16. As a user, I want to submit photos of my sneakers in-app so that I can get an estimated resale price range.
17. As a user, I want to pay a monthly fee so that I get early access to newly arrived inventory before it goes public.
